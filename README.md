# PulseFi AI Fintech Dashboard

A modern single-page fintech dashboard prototype with:

- AI insights + AI summary panel
- AI expense tracker charts
- AI finance chat
- Bill scanner (OCR flow placeholder)
- Paytm import flow placeholder
- Google login button + signup/login actions
- Biometric toggle UX
- Smooth, high-refresh friendly UI animations

## Run locally

```bash
python3 -m http.server 4173
# open http://localhost:4173
```

## Open-source AI model plan

- **LLM:** Llama 3.1 / Mistral via Ollama
- **OCR:** Tesseract OCR
- **Embeddings:** nomic-embed-text
- **Charting:** native Canvas rendering

> This repo is currently a frontend prototype with local/mock AI behavior in `app.js`.
